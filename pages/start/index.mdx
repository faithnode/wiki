# Как начать играть?

**Приобретите проходку**

<Figure title="Приобрести проходку возможно только на нашем сайте  ">
    ![](/assets/start/buyingpass.webp)
</Figure>

**Установите актуальную версию Minecraft**

На сервере можно играть с любого Minecraft лаунчера, но мы рекомендуем играть с [официального, линцензионного лаунчера](https://www.minecraft.net/download "_blank").
Также рекомендуем играть с нашей [сборкой модов](/assets/files/assembly/сборка.zip). Подробнее о разрешенных,
запрещенных и рекомендуемых модах можно прочиать на [этой](/wiki/addons/mods) странице


**Подключитесь к серверу**


**IP-Адрес сервера**

```
mc.faithcraft.su
```

**IP-Адрес творческого сервера**

```
creative.faithcraft.su
```
