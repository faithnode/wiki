# CRON: Автоматизация задач

<ColorBlock color="yellow">
    [CRON]({{ links.cronWikiUrl }} "blank") - это утилита-планировщик задач в Unix-подобных операционных системах, которая позволяет автоматически выполнять скрипты, команды или программы в заданное время.
</ColorBlock>

**Синтаксис CRON:**

```
* * * * *
│ │ │ │ │
│ │ │ │ └─── день недели (0-6) (воскресенье = 0)
│ │ │ └───── месяц (1-12)
│ │ └─────── день месяца (1-31)
│ └───────── час (0-23)
└─────────── минуты (0-59)
```
