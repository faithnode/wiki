# Плагины

<Details summary="Кастомные стойки для брони">
    Теперь стойкам можно придавать самые разные позы из Bedrock версии игры, что расширяет возможности для создания
    декоративных элементов.
    Кроме того, плагин позволяет давать в руки абсолютно любые предметы, что открывает новые возможности для реализации
    идей.
    Чтобы изменить позу стойки, зажмите Shift и нажмите ПКМ на стойку.

    <Figure title="">
        ![arnourstand.webp](/assets/gameplay/arnourstand.webp)
    </Figure>
</Details>

<Details summary="Блоки - головы">

    Этот плагин позволяет игрокам надевать на свою голову некоторые предметы, перетаскивая их на соответствующий слот
    выбора снаряжения для головы . Таким образом, игроки могут индивидуализировать свой вид в игре, используя любые
    предметы, которые они хотят. Это может быть полезно для создания уникального образа или просто для добавления
    уникальности в игровой процесс.

    <Figure title="">
        ![blockhead.webp](/assets/gameplay/blockhead.webp)
    </Figure>
</Details>

<Details summary="Еда">
    Этот плагин позволяет игрокам есть еду, даже если у них уже полоска голода полностью заполнена. Кроме этого, при
    использовании вашего плагина, будет пополняться невидимая полоска сытости, которая позволит игрокам регенерировать
    здоровье быстрее. Это может быть полезно во время сражений или когда игроки хотят быстрее восстановить свое
    здоровье.

</Details>
