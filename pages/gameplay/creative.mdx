# Креатив сервер

Креатив сервер – наш вспомогательный сервер, открытый всем вашим идеям. Стройте, эксперементируйте, проектируете – огромный простор для ваших фантазий.

Попасть на сервер можно двумя способами:
- Подключиться, используя IP-Адрес сервера
```
creative.faithcraft.su
```
- Подключиться, используя каоманду на нашем основном сервере
`/creative` или `/server creative`

Также на сервере для удобства присутсвуют несколько миров, перемещаться по которым можно через порталы в центральной хаб-локации, попасть в которую можно прописать команду `/hub`.
