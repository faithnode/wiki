# Основные команды сервера

`/help` - Узнать все доступные вам команды

`/msg` `/pm` `/tell` - Отправить личное сообщение игроку

`/r` - Ответить игроку на личное сообщение

`/skin` - Выбрать скин из заранее загруженных в вашем профиле на наешм сайте

`/sit` - Сесть

`/lay` - Лечь

`/co i` - Просмотреть логи конкретного блока

`/co near` - Просмотреть логи близлежайшей области

`/server creative` - Перемеситься на наш креатив сервер

`/me` `/try` `/do` - РП команды, подробнее о которых можно прочитать вот [тут](/wiki/gameplay/roleplay)
