# {{ metadata.title }}

---

Для стабильной работы сервера (особенно с модами и плагинами) необходимо использовать строго определенную версию Java. Хотя чистая (Vanilla) игра часто поддерживает диапазоны («Java X и новее»), **модифицированные ядра требуют конкретную версию**.

Использование неверной Java приведет к ошибке `UnsupportedClassVersionError` или моментальному крашу сервера при запуске.

## Сводная таблица версий

| Версия Minecraft | Поддерживаемый диапазон | Рекомендуемая КОНКРЕТНАЯ версия Java |
| :--- | :--- | :--- |
| **26.1 и новее** | Java 25 и выше | **Java 25** |
| **1.20.5 — 1.21.x** | Java 21 и выше | **Java 21** |
| **1.18 — 1.20.4** | Java 17 и выше | **Java 17** |
| **1.17 — 1.17.1** | Java 16 и выше | **Java 16** |
| **1.12 — 1.16.5** | Java 8 и выше | **Java 8** *(Критично для Forge/Fabric)* |
| **1.6.1 — 1.11.2** | Java 6 и выше | **Java 8** *(Для стабильности плагинов)* |
| **pre-Classic — 1.5.2** | Java 5 и выше | **Java 8** *(Старый код несовместим с Java 9+)* |
